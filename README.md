# Mosera Inventory Manager (Kotlin)

## Overview

This software is a console-based inventory management system designed for **Mosera Automobile**. It allows users to manage a list of car parts, track their pricing, and filter items by category. 

I developed this project to demonstrate proficiency in the Kotlin programming language, specifically focusing on its modern syntax and safety features.
